Projects

Quest Log

A passion project for a video game discovery app where users can read top reviews and the latest news. Includes a Tinder-like system for users to receive tailored content based on games they already like along with helping them find new ones. Users are also able to create and rate reviews alongside professionals allowing for more trustworthy content. In all, Quest Log aims to be a single source of truth for all things gaming.
